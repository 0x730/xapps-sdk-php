# Contributing

## Scope

This repository is a public distribution surface derived from the private canonical `xapps` engineering repo.

That means:

- public packages and examples are accepted here
- operational core, gateway, deploy, and private reference lanes are not maintained here

## Before opening changes

Please:

1. open or review an issue first for non-trivial work
2. keep changes narrowly scoped
3. include tests or smoke validation where practical
4. avoid unrelated formatting churn

## Release note expectation

If a change affects a published package surface, include a short release-note summary in the PR description.
