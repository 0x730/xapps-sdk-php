# xapps SDK PHP

Public PHP SDK distribution repo for Xapps.

Current published packages:

- `xapps-platform/xapps-php`
- `xapps-platform/xapps-backend-kit`

Quick package guide:

- `xapps-platform/xapps-php`: lower-level PHP SDK for gateway, callback, payment-return, and publisher-admin flows.
- `xapps-platform/xapps-backend-kit`: higher-level PHP backend kit that sits above `xapps-platform/xapps-php`.

Install examples:

```bash
composer require xapps-platform/xapps-php
composer require xapps-platform/xapps-backend-kit
```

Canonical engineering source remains private. This repo is the public distribution surface under:
